# Mercado Libre DataSec Challenge

Este repositorio contiene las soluciones para el Desafío de Datos de Mercado Libre.

---

## Challenge 1 — Minesweeper Number of Neighbouring Mines

**Archivo:** `solution_minesweeper.py`

Crea una función que, dado un tablero de Buscaminas representado como una lista bidimensional, devuelve otro tablero con la cantidad de minas vecinas en cada casilla.

---

## Challenge 2 — REST API: Best TV Shows in Genre

**Archivo:** `solution_best_in_genre.py`

Implementa una función que consulta una API paginada para encontrar la serie de TV con la calificación más alta en IMDb dentro de un género determinado.

---

## Challenge 3 — SQL: Advertising System Failures Report

**Archivo:** `applicant_query.sql`

Devuelve todos los clientes con más de 3 eventos de fallo, ordenados por número de fallos.

---

## Challenge 4 — Go CLI: Text Summarizer with GenAI

**Archivo:** `solution_summarizer.go`

Desarrolla una aplicación de línea de comandos en **Go** que resume el contenido de un archivo de texto utilizando una API pública de **GenAI**.

---

## ⚙️ Entorno
